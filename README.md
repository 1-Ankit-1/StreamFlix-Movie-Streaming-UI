# 🎬 StreamFlix – Movie Streaming Platform UI

StreamFlix is a modern movie streaming frontend inspired by platforms like Netflix and Prime Video.

Built using **HTML**, **CSS**, and **JavaScript**, this project focuses on responsive layouts, cinematic UI design, hover animations, and interactive movie sections.

Perfect for frontend portfolio projects and UI design practice.

---

# ✨ Features

- 🎥 Netflix-Inspired UI
- 📱 Fully Responsive Design
- 🎬 Hero Banner Section
- 🍿 Trending Movie Cards
- 🔍 Search Bar UI
- ❤️ Watchlist Button
- ⚡ Smooth Hover Effects
- 🌙 Dark Theme Interface
- 🎨 Modern Streaming Layout
- 📺 Category Sections
